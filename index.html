To view the source code:

Right click on the page and choose view Page source. Enable web browsers web development tools

Web tools on Firefox:

You can access the tools by clicking on the burger menu icon and then choosing Web Developer and after that toggle tool. Try finding the following

  • Find element
  • Style editor
  • Console

Web tools on Chrome:

Chrome web developer tools can be found under three vertical dots, then selecting more tools and finally selecting Developer tools. Try finding the following

  • Select an element
  • Body after selecting an element
  • Console

Text editors

Sublime is recommended text editor, it is easy to extend by clicking ctrl+shift+P and choosing package control install. It also comes as portable version
